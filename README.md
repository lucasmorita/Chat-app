# Objective 
This is a project for learning, so I designed a webchat that uses WebSockets for real time messages. 

## Stack
For the frontend, I'm using:
- React
- Vite
- Tailwind with DaisyUI

For the backend:
- Ktor
- PostgreSQL
- Planning to use Cassandra for storing messages
